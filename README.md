## 🚀 **Quick Start Guide**

### ⚡ **One-Command Setup**
```bash
# Clone and start everything
git clone <repository-url>
cd MealMatch
npm run setup-all    # Installs all dependencies
npm run dev-all      # Starts both client and server
```

### 🔧 **Manual Setup**
```bash
# 1. Clone repository
git clone <repository-url>
cd MealMatch

# 2. Environment Configuration
# Copy example environment files
cp client/.env.example client/.env
cp server/.env.example server/.env

# 3. Configure API Keys
# Edit client/.env and add your Weather API key:
# REACT_APP_WEATHER_API_KEY=your_weather_api_key_here
# Get free API key from: https://www.weatherapi.com/

# 4. Backend setup
cd server
npm install
npm run dev

# 5. Frontend setup (new terminal)
cd client  
npm install
npm start

# 6. Access application
🌐 Frontend: http://localhost:3000 (development) / https://mealmatch-frontend.onrender.com (production)
🔧 Backend API: http://localhost:5000 (development) / https://mealmatch-backend.onrender.com (production)
```

### 🔐 **Environment Variables Setup**

#### **Client (.env)**
```bash
# Weather API Configuration (Required for weather features)
REACT_APP_WEATHER_API_KEY=your_weather_api_key_here
REACT_APP_WEATHER_API_BASE_URL=https://api.weatherapi.com/v1/current.json

# API Base URL
REACT_APP_API_BASE_URL=https://mealmatch-backend.onrender.com/api
```

#### **Server (.env)**
```bash
# MongoDB Configuration
MONGODB_URI=mongodb://localhost:27017/mealmatch

# Server Configuration  
PORT=5000
NODE_ENV=development

# JWT Secret (optional)
JWT_SECRET=your_jwt_secret_here

# CORS Configuration
CLIENT_URL=https://mealmatch-frontend.onrender.com
```

#### **Getting Weather API Key**
1. 🌐 Visit [WeatherAPI.com](https://www.weatherapi.com/)
2. 📝 Sign up for a free account
3. 🔑 Get your API key from the dashboard
4. 📋 Copy the key to `client/.env` file
5. ✅ Weather features will work automatically!
